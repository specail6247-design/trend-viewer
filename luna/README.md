# 🌙 루나 릴스 스튜디오

**Insta360 Luna Ultra 팁 릴스를 브라우저에서 바로 만드는 도구**

- 📋 팁 릴스 기획안 12개 — 장면별 촬영 가이드 + AI 영상 생성용 프롬프트 + 자막 타이밍 + 인스타 캡션/해시태그
- 🎬 릴스 에디터 — 클립 업로드 → 자막·타이틀 자동 합성 → 1080×1920 세로 mp4 렌더링 (약 1분 내외)
- 🚀 홍보 가이드 — 업로드 전략, 체크리스트, 해시태그 세트

## 사용법

1. **기획안 탭**에서 마음에 드는 팁을 골라 촬영 (또는 AI 프롬프트로 영상 생성)
2. **"이 기획안으로 만들기"** 클릭 → 자막이 자동으로 채워진 에디터로 이동
3. 촬영한 클립 업로드 → **⚡ 자동 배분**으로 시간 맞추기 → **🎬 릴스 만들기**
4. 완성된 mp4를 인스타그램 릴스에 업로드 (트렌딩 오디오는 인스타 앱에서!)

모든 영상 처리는 브라우저 안에서만 이루어지며 어디에도 업로드되지 않습니다.

## 기술

- 순수 HTML/CSS/JS, 빌드·서버·외부 라이브러리 없음
- Canvas 합성 + MediaRecorder(H.264 mp4)로 렌더링
- GitHub Pages 정적 호스팅
